# Bonjour GitHub

Projet du module 13 réalisé par **Cedric Nguena**.

## Description
Ce projet démontre l'utilisation de GitHub avec :
- Une page web statique (`index.html`)
- Une licence MIT (`LICENSE`)
- Documentation en Markdown
- Gestion des tâches avec Issues
- Hébergement avec GitHub Pages

Lien du site : [à ajouter après activation GitHub Pages](https://cedric-nguena.github.io/bonjour-github/)
